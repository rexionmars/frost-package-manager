## Frost Package Manager 📦

⚠️Attention this project is still under constant development, this entire guide may be changed.

## Overview
- [x] Native
- [x] Fast
- [x] Portable
- [x] Linux
- [ ] Windows
- [ ] MacOS

Prerequisites for FrostPKG
- [x] Tar >= 1.34
- [x] Bash >= 4.4.18
